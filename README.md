## Project Overview
**Task-Tracker** is a **single-page web application** that functions as a **To-Do List**. Users can **add, complete, and remove tasks**, and the task list updates dynamically without page reload.  

This project demonstrates **front-end development skills**, **DOM manipulation with JavaScript**, responsive design, accessibility, and a **dark/light theme toggle**.

## Features
- Add new tasks with input validation  
- Mark tasks as completed  
- Delete individual tasks or clear all tasks  
- **Dark / Light Theme toggle**  
- Responsive design for mobile, tablet, and desktop  
- Smooth and intuitive UI

## Technologies Used
- **HTML5** – Semantic structure  
- **CSS3** – Flexbox/Grid layout, media queries  
- **JavaScript (ES6+)** – DOM manipulation, event listeners, loops, conditionals  
- **Git & GitHub** – Version control with meaningful commits  
- **Cloud Hosting** – Deployed online
